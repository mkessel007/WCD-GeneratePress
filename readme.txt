Add line height typography to the customizer controls of the GeneratePress Theme

== Description ==
Add line height typography to the customizer controls of the GeneratePress Theme. Requires the premium extension to add line height controls to heading tags.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the \'Plugins\' screen in WordPress

== Frequently Asked Questions ==
= Do I need the Premium Plugin GeneratePress AddOn? =

No, however you can only set the line height globally via the body without the addon

= I get an invalid value error? =

You need to ensure you use a compatible format such as em, rem, px and %
